# 📄✨ ASIF Subtitle: Advanced Subtitle Improvements and Fixes

As if the word would never resolve&hellip;&trade;

> [!CAUTION]
>
> This repository is publicly available. However, it is a work in progress and not yet ready. Please do not use it before it is ready to use.

ASIF Subtitle (or ASIF, stands for Adanced Subtitle Improvements and Fixes) is a tool to create readable and accessible captions and fix captions. It can also split transcription or captions into words without sacrificing readability and creating single-word captions.

It aims to:

- Redefine readability
- Completely reduce and even eliminate single-word captions
- Fix *sloptitles*[^1] or *autocraptions*[^2]
- Implement a better splitting
- Make a better alternative to the *freaking* single-word caption or Hormozi-style caption generator
- Create a readable caption for a vertical video (It's not impossible to balance between aesthetic and readability)
- and so on

[^1]: poorly AI-generated subtitles, Nakhwan Maulana's term, its project creator
[^2]: poorly auto-generated captions, coined by Meryl Evans

> [!NOTE]
> 
> Those caption results may be imperfect. Please edit those to fix more problems.
